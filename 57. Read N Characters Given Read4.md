```python
class Solution:
    def read(self, buf, n):
        """
        :type buf: Destination buffer (List[str])
        :type n: Number of characters to read (int)
        :rtype: The number of actual characters read (int)
        """
        index = 0
        while n > 0:
            buf4 = [""]*4
            l = read4(buf4)
            if not l:
                return index
            for i in range(min(l, n)):
                buf[index] = buf4[i]
                index += 1
                n -= 1
        return index
```
Time complexity: O(N)  
Space complexity: O(N)
